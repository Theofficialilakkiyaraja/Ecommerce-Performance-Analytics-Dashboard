# E-Commerce Performance Analytics Dashboard (Excel)

## Project Overview
This project demonstrates an interactive Excel dashboard designed to analyze e-commerce sales performance.

The dashboard analyzes:
- 25K+ orders
- 178K units sold
- $12.6M in sales
- $1.46M in profit

## Key Features
- KPI tracking (Sales, Profit, Orders, Quantity)
- Sales trend analysis
- Regional performance insights
- Product profitability analysis
- Discount impact analysis
- Returns monitoring

## Tools Used
- Microsoft Excel 2019
- Pivot Tables
- Pivot Charts
- Slicers
- Data Modeling
